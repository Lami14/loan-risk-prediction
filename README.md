# 💳 Loan Risk Prediction System

## 📌 Problem
Banks need to determine whether a customer qualifies for a loan based on financial history.

## 💡 Solution
This project uses a machine learning model to predict loan approval based on user financial data.

## 🛠️ Tools Used
- Python
- pandas
- scikit-learn
- Flask

## 🚀 Features
- Train ML model on financial data
- Predict loan approval
- REST API for real-time predictions
